    What is 'mose'?

    'mose' is a visualization tool for iRacing car setup file. It's running as Web application on your PC. You can see the difference between two or more setup files simultaneously, quickly and visually. In addition, mose can track the lap time marked by the setup that you want to record with it.

    Concept

    For beginners, it's really tough to create good setup because they have to learn a lot of theory and also plenty of experience by their own is needed. On the other hand, there are some nice setups created by talented and/or experienced drivers on the forum week by week. That is chance to learn what difference makes different speed and/or handling of the car. mose assists this process.

    Install

    1. Install Perl. I recommend Strawberry Perl.
    2. Download and extract 'mose'.
    3. Double-click 'setup.bat' to install additional Perl modules.
    4. Open and edit 'mose.conf' to set 'setupdir' properly.

    Uninstall

    Delete 'mose' folder.

    Usage

    Double-click 'mose-run.bat', then open http://localhost:3000 in your favorite browser. If you want to change port number, open and edit 'mose-run.bat'.

    Supported Car

    Currently, Chevrolet Monte-Carlo only.
